# ELT vs ETL using Apache Druid, Neon PostgreSQL and Superset

## Project Overview

This project demonstrates the difference between ETL (Extract, Transform, Load) and ELT (Extract, Load, Transform) using Apache Druid, Neon PostgreSQL, and Apache Superset.

## Architecture
CSV Dataset
     ↓
Neon PostgreSQL
     ↓
Apache Druid
     ↓
Apache Superset
     ↓
Analytics Dashboard


## Technologies Used

* Apache Druid
* Neon PostgreSQL
* Apache Superset
* Docker
* Python
* SQL

## Features

* Data ingestion into Apache Druid
* ELT workflow implementation
* SQL-based transformations
* Real-time analytics
* Dashboard creation in Superset

## Data Analysis Performed

* Total electricity delivered
* Total revenue generated
* Daily revenue analysis
* Payment method distribution
* Station performance analysis

## Apache Druid

Apache Druid is a high-performance analytics database optimized for real-time analytics and time-series workloads.

## Apache Superset

Apache Superset is an open-source business intelligence and visualization platform used to create dashboards and charts.
